HTML Massage

  • correct HTML
  • convert HTML to XHTML automatically
  • compact HTML
  • pretty print HTML
  • writes links relative
  • write HTML with hooks for Section 508 compliance

Options

java tool.html.Massage [-xhtml|-prettyprint|-compact] [-prettyprint] [] []
  • -prettyprint indents according to the nesting level of the tag in the parse tree
  • -xhtml -- writes XHTML compatible output (tags balanced, attribute values quoted, terminated empty tags, "name"=>"id", no ampersand in attribute values, and so on). See XHTML 1.0 Spec)
  • -relative base-URL -- make hyperlinks relative to base-URL, which is directory which corresponds to the top of the web site
  • -usability checks for like no ALT on IMG, no TITLE on tables, and other things that hinder usability for some. See Web Content Accessibility Guidelines 1.0.
  • -perfect or -all turns on all of the above
  • -stdout renders transformed page to text screen instead of rewriting disk file
  • -verbose lists files as they are rewritten, lists linked-to pages as they are signed, and shows the signature or reason the page was not signed (host was not available, protocol was news, mailto, telnet, javascript). -verbose is on by default.
  • -quiet is the opposite of -verbose
  • -version write the version of this class and immediately exits
