Robust Hyperlink Signer

Automatically rewrite HTML pages to make their hyperlinks robust. Recurses directories and rewrites links in files named ".html" and ".htm" to make their links robust. The page rewriting only touches links — all other text is left untouched. See Robust Hyperlinks web site for details.

Options

java tool.html.Robust [options] URL [filename]
  • URL is taken as the base URL
