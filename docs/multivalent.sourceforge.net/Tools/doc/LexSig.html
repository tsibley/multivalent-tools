Lexical Signature

Generate a lexical signature for given document. See Robust Hyperlinks web site for details.

Options

java tool.doc.LexSig [options] URL-or-file
  • -verbose — display computations. Default: on, since word frequency gathering can be lengthy.
  • -quiet — opposite of -verbose

See Also

To automatically rewrite HTML pages to use Robust Hyperlinks, see tool.html.RobustSigner.
