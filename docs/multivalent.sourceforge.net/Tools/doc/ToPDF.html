Convert to PDF

This tools was written as an example for developers. It is not meant for end users, who are referred to Acrobat for this task.

Translates any supported document format (HTML, DVI, man pages, ...) to PDF. any doc and builds doc tree, traverse document tree and pull out data. also example of writing PDF with PDFWriter library

Options

java tool.doc.ToPDF [options] document-file(s)
  • -paper paper-size -- paper size (Default: US letter.)
  • -border length -- size of border between content and page, around all four sides (top, bottom, left, right) (Default: 1 inch)
  • -page range -- selected pages from source PDF, for those formats that are paginated
